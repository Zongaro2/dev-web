Para o que serve cada ferramenta:
O python pode ser usado como uma linguagem que dará suporte pra fazer
o código como um todo, O sistema de versionamento de código do Git pode 
ser usado para slvar várias versões de nossos trabalhos e salvar
os nossos trabalhos e projetos, O VSCODE será a nossa IDE que usaremos
para editar e escrever os nossos código, o SQLITE será usado para
gerenciar os bancos de dados, e os navegadores serão usados para
pesquisa e como uma forma de ajudar e nos guiar no curso como um todo.
Com isso tudo a aula será mais dinâmica.
